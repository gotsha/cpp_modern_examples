# Einheitliche Initialisierung

Es lassen sich mit geschweiften Klammern alle möglichen Arten einer Variablen-Initialisierung (auch Objekte)
mit einer einheitlichen Schreibweise initialisieren.

Die Beispiele aus dem Verzeichnis gehen auf folgende Einzelfälle ein:

  * Variablen elementaren Datentyps
  * Dynamisch allokierte Felder
  * Statisch allokierte (bzw. definierte) Felder
  * Objekte zu C++ Standard-Klassen
  * Objekte von benutzerdefinierten Klassen mit `public`-Attributen

Ferner wird auf die Initialisierung von Default-Werten eingegangen.
